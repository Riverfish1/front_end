# 警务前端

## Getting Started

#### 克隆

`$ git clone https://git.oschina.net/nc-dev/font_end.git`

#### 进入目录

`$ cd font_end`

#### 安装依赖

`$ npm i`

#### 启动

`$ npm run dev`

	然后在浏览器中打开localhost:8989

#### 代理

* 打开 switchOmega , 开启代理。
